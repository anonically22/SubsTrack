# SubsTrack: Institutional Subscription Management

SubsTrack is a high-fidelity psychological subscription tracking platform designed for institutional-grade financial oversight. It provides a clinical, analytical interface to identify and control silent capital leaks.

## Design Aesthetic: Variant 2

The current implementation follows an **Institutional Financial** aesthetic, characterized by:

- **Structured Grid System**: An 80x80px grid background providing a rigid structural foundation.
- **Cinematic Typography**: 
  - `Inter` for technical UI and display statements.
  - `Noto Serif Display` for emotional, high-impact statements.
  - `Noto Sans Mono` for precise financial data.
- **Micro-Animations**: Animated financial counters and pulsing "Operational" indicators.
- **Clinical Clarity**: A monochromatic dark palette (`#0e0e0e`) with primary clinical blue (`#337de6`) accents.

## Core Frameworks

- **React + Vite**: Performance-first frontend architecture.
- **Tailwind CSS v3**: Clean, utility-first styling for structural rigidity.
- **Framer Motion**: Subtle, controlled transitions and interactive metrics.
- **Lucide React**: Minimalist iconography for functional clarity.
