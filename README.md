# SubsTrack

This is my personal financial hub. Originally, it was just an idea to track my subscriptions, but as I built it, I realized the real problem I was solving was **financial awareness**. 

Most apps focus on the mechanics of budgeting—categorizing every coffee or rent payment. I wanted something different. I wanted to **see the patterns**. I wanted to understand how small, frictionless recurring payments—like that ₹499/month streaming sub—actually shape my long-term liquidity.

### The Vision
SubsTrack is built for me, but designed to institutional standards. It’s calm, structured, and analytical. It doesn't tell me what to do; it just surfaces the data cleanly enough that I can make my own decisions.

### Core Stack
- **Frontend**: React + Vite (Fast, modular, precise)
- **Styling**: Tailwind CSS (Minimal, utility-first)
- **Animations**: Framer Motion (Subtle micro-interactions)
- **Backend**: Supabase (Real-time data synchronization)

### Key Features
- **Ledger Management**: Chronological tracking of income and expenses.
- **Fleet Overview**: High-density subscription management.
- **Behavioral Insights**: Automated observations on spending growth and patterns.
- **Privacy First**: Built on top of Supabase with robust RLS policies.
- **PWA Support**: Installed on my home screen for immediate access.

### Why the "Institutional" Look?
I intentionally avoided the "friendly" or "playful" look of most modern apps. I wanted this to feel like a high-end data platform. Monospaced typography for financial values, wide gutters, and a strict grid system—everything is designed to treat my finances with the precision they deserve.
